# Calculadora de Índice de Masa Corporal (IMC)

Este proyecto consiste en una aplicación interactiva de consola desarrollada en Python. El programa solicita datos personales esenciales del usuario, implementa un sistema robusto de validación para garantizar que la información ingresada sea correcta y calcula el Índice de Masa Corporal (IMC) utilizando estándares de salud pública.

## Características del Proyecto
* **Captura de Datos Avanzada**: Almacena de forma independiente el nombre, los apellidos, la edad, el peso y la estatura en variables distintas.
* **Validación Robusta contra Errores**: Bloquea campos vacíos y evita que el usuario introduzca caracteres no numéricos o valores negativos en campos clave.
* **Interfaz Dinámica**: Hace uso de códigos de color ANSI en la terminal para separar visualmente las preguntas, las alertas de error y los resultados.
* **Diagnóstico Personalizado**: Evalúa el IMC e imprime un resumen amigable dirigiéndose al usuario por su nombre y haciendo mención de su edad cronológica.

## Cómo Ejecutar el Programa
1. Asegúrate de tener instalado Python 3.11 o superior en tu equipo.
2. Descarga o clona este repositorio en tu computadora.
3. Abre una terminal o consola de comandos en la carpeta del proyecto.
4. Ejecuta el archivo con el siguiente comando:
   ```bash
   python Calculadora_indice_masa_corporal__Proyecto_1_.py
   ```
